# Native Navigation Matrix

## macOS
- Primary layout: `NavigationSplitView` (sidebar + content + optional inspector)
- Toolbar: primary actions (new, search, share)
- Multiwindow support for notes/files/websites
- Context menus for list actions

## iPadOS
- Primary layout: `NavigationSplitView` (2-3 columns)
- Drag and drop between panes
- Toolbar actions in top bar
- Optional inspector for metadata/details

## iOS
- Primary layout: tab bar for top-level sections
- `NavigationStack` for detail flow
- Sheets for secondary actions
- Swipe actions for list row operations
