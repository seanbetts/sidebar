# Native API Inventory

## Core Frameworks
- SwiftUI: UI layer, navigation, state, accessibility
- Combine: async bindings (optional with async/await)

## Viewing / Media
- Quick Look: default file preview
- PDFKit: PDFs
- AVFoundation / AVKit: audio/video
- WebKit: HTML fallback
- LinkPresentation: link previews (optional)

## System Services
- CoreLocation: weather + places
- UniformTypeIdentifiers: file types
- PhotosUI / UIDocumentPickerViewController: file import
- OSLog: logging

## Platform UX
- macOS: AppKit integration for toolbar/menu/commands where needed
- iOS: SFSafariViewController for external web views

## Data / Sync
- Supabase Swift SDK: auth + realtime (if used)
